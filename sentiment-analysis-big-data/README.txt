DATSci7012 – Social Media Streaming & Batch Analytics    

Included Files
- Music Sentiment Analysis Report.pdf – Main report
- Batch_Analysis.ipynb – YouTube batch pipeline, Spark cleaning, sentiment analysis
- Streaming_Analysis.ipynb – Reddit live streaming pipeline using MongoDB
- Realtime_Dashboard.ipynb – Interactive dashboard with sentiment charts & word clouds
- comments.jsonl – Raw dataset collected using YouTube API

Running Notes
- Requires Python 3.8+, PySpark, MongoDB, PRAW, and NLTK with VADER lexicon downloaded.
- Streaming notebook requires active Reddit API credentials and MongoDB running locally.
- Dashboard notebook uses ipywidgets and wordcloud packages.
